USANDO LINKS

Links Externos

Um link externo é uma ligação com alguma página da internet que não pertence à este site. Exemplo: página do youtube do canal curso em vídeo

No entanto, ao utilizar um link dessa forma, vai haver um redirecionamento para a página apontada pelo link, o que faz o usuário sair do site. Para evitar isso, é necessário usar alguns parâmetros, como o 'target' e o 'rel'. Exemplo: novo link para curso em vídeo

Além dos links externos também existem os links internos, ou seja, aqueles que apontam para outras páginas do próprio site. Exemplo: link para a segunda página do site

Se, na organização do seu projeto, você tiver links que apontam para pastas internas, é necessário, neste caso, colocar o endereço completo do arquivo, ou seja, digitar tanto o nome da pasta quanto o nome do arquivo, separando-os por uma barra diagonal. Exemplo: link para página em outra pasta

Link para Download

Links para download envolvem parâmetros de configuração de mídia, ou seja, parâmetros específicos para baixar arquivos de diferentes naturezas, como texto, audio, video, etc. Exemplo: link para download de pdf
